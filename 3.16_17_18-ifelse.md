## goto 条件分支
汇编中可以使用jmp命令进行类似`goto`的跳转实现条件分支 , 例如:
```c
long gotodiff_se(long x, long y)
{
    //这种C代码可以直接逐字翻译为汇编
    long result;
    if (x >= y)     //使用 cmpq 和 jge
        goto X_ge_y;
    lt_cnt++;       //这里使用了(%rip) , 这是一个指令指针寄存器
    result = y - x;
    return result;
X_ge_y:
    ge_cnt++;
    result = x - y;
    return result;
}
```
上面第一个语句块 `lt_cnt++;  result = y - x;  return result;`属于` x < y ` , 下面属于`x >= y`
